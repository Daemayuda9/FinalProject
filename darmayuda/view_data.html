VISI & MISI D3 TEKNIK INFORMATIKA

VISI Program Studi D3 Teknik Informatika

Tahun 2030 menjadi Program Studi Teknik Informatika Kelas Dunia yang Unggul dalam Bidang Web Development, Computer Network, dan 2D Animation yang berbasis enterpreneurship.

MISI Program Studi D3 Teknik Informatika

  • Menyelenggarakan pendidikan yang berkualitas di bidang Web Development, Computer Network, dan 2D Animation yang berbasis enterpreneurship.
  • Melaksanakan penelitian dan pengembangan di bidang Web Development, Computer Network, dan 2D Animation yang memberikan kontribusi pada ilmu pengetahuan dan teknologi.
  • Melaksanakan pengabdian kepada masyarakat di bidang Web Development, Computer Network, dan 2D Animation untuk meningkatkan kualitas hidup masyarakat.
  • Mengembangkan jejaring kerjasama dengan industri dan komunitas untuk meningkatkan relevansi kurikulum dan peluang karir mahasiswa.
